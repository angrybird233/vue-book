# 经销商人员删除

开发者：冯俊杰

## 接口地址

`/dealer/v1/admin/personnel/delete`

## 请求方式

`POST`

## 请求参数

| 字段       | 类型  | 必填 | 描述         |
| ---------- | ----- | ---- | ------------ |
| personnel_id   | int    | 是   | 经销人员ID |

## 成功响应

| 字段       | 类型   | 描述       |
| ---------- | ------ | ---------- |
| status     | string | 状态       |
| httpCode   | int    | http状态码 |
| statusCode | int    | 状态码     |
| data       | array  | 返回数据   |

## 失败响应

| 字段       | 类型   | 描述       |
| ---------- | ------ | ---------- |
| status     | string | 状态       |
| httpCode   | int    | http状态码 |
| statusCode | int    | 状态码     |
| message    | string | 错误信息   |

## 返回结果预览

```json
{
    "status": "success",
    "httpCode": 200,
    "statusCode": 0,
    "requestId": "API_90198db104de7695df1e6f7e8979eb42",
    "data": {}
}
```
