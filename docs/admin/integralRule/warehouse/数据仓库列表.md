# 数据仓库列表

开发者：王龙飞

## 接口地址

`/integralRule/v1/admin/warehouse/list`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| warehouse_id | int | 否 | 仓库编码 |
| name | string | 否 | 仓库名称 |
| page   | int    | 否  | 页码   |
| page_size   | int    | 否  | 条数   |

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

### data参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| list | array | 数据列表 |
| meta | array | 分页信息 |

#### list参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| warehouse_id | int | ID |
| name | string |  名称 |
| remark | string |  备注 |

### meta参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| current_page | int | 当前页码 |
| last_page | int | 总页数 |
| page_size | int | 每页条数 |
| total | int | 总条数 |
