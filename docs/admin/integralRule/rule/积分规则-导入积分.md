# 积分规则 - 导入积分

开发者：王龙飞

## 模板下载地址
`/admin/download/template`  `GET`

## 模板下载请求参数
| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| filename   | string    | 是   | 模板文件名称：值必须是 ”积分导入模板.xlsx“   |

## 接口地址

`/integralRule/v1/admin/ruleDetail/integralImport`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| rule_code | string | 是 | 规则编码 |
| file | file | 是 | 文件 |

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

### data参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| success_count | int | 成功条数 |
| err_count | int | 失败条数 |
| err_data | array | 错误数据列表 |
| header | array | 错误数据列表表头信息 |

### err_data

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| bind_user | string | 经销商编码 |
| add_integral | string | 增加的积分 |
| reduce_integral |string | 减少的积分 |
| remark | string | 备注 |
| err_msg | string |  错误信息 |
