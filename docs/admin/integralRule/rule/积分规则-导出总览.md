# 积分规则 - 导出总览

开发者：王龙飞

## 接口地址

`/integralRule/v1/admin/ruleDetail/overviewExport`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| rule_code | string | 是 | 规则编码 |
| bind_user | string | 否 | 经销商编码 |
| bind_user_name | string | 否 | 经销商名称 |

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |
