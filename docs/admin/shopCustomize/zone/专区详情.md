# 专区详情
开发者：王龙飞

## 接口地址
`/shopCustomize/v1/admin/zone/detail`

## 请求方式
`POST`

## 请求参数
| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| zone_id | int | 是 | 专区主键 |


## 成功响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

### data参数
| 字段 | 类型 | 描述 |
| --- | --- | --- |
| zone_id | int | 主键 |
| zone_name | string | 专区名称 |
| zone_type | int | 专区类型,1单横幅，2双横幅，3横向商品列表 |
| img_url | string | 图片 |
| goods_group_ids | array | 关联的商品组ID集合 |
