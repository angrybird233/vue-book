# 分类推荐详情
开发者：王龙飞

## 接口地址
`/shopCustomize/v1/admin/recommendClass/detail`

## 请求方式
`POST`

## 请求参数
| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| recommend_class_id | int | 是 |  banner主键 |


## 成功响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

### data参数
| 字段 | 类型 | 描述 |
| --- | --- | --- |
| recommend_class_id | int | 主键 |
| recommend_class_name | string | 名称 |
| img_url | string | 图片 |
| cate_one | int |  一级分类主键 |
| cate_two | int |  二级分类主键 |
