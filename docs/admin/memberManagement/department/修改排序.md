# 组织排序接口

开发者：王俊锋泽

## 接口地址

`/member/v1/admin/department/changeOrder`

## 请求方式

`POST`

## 请求参数

|参数|类型|是否必填|说明|
| --- | --- | --- | --- |
| dpId | string | 是 | 组织id |
| dpDisplayorder | int | 是 | 排序值 |

## 返回参数

```json
{
  "status": "success",
  "httpCode": 200,
  "statusCode": 0,
  "data": {}
}
```
