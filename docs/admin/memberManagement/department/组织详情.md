#组织详情接口

开发者：王俊锋泽

## 接口地址
`/member/v1/admin/department/editQuery`

## 请求方式
  `POST`

## 请求参数

|参数|类型|是否必填|说明|
| - | - | - | - |
| dpId | int | 是 | 组织id |


## 返回参数
|参数|类型|说明|
| - | - | - |
| dpName | string| 名称 |
| dpParentid | string| 上级部门id |
| dpParentName | string| 上级部门名称 |
| dptId | string |  组织类型ID |
| dpLevel | int | 组织层级 |
| dpSerialNum | string |  组织编号 |
| dpLeadList | object | 部门领导信息 |

## dpLeadList 返回参数
|参数|类型|说明|
| - | - | - |
| face | string | 头像url |
| name | string | 姓名 |
| uid | string | 用户ID |
