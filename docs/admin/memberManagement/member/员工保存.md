# 员工保存接口

开发者：王俊锋泽

## 接口地址

`/member/v1/admin/member/save`

## 请求方式

`POST`

## 请求参数

| 参数        | 类型   | 说明 | 示例 |
| ----------- | ------ | ---- | ---- |
| uid     | string | uid 编辑时传递|  E18DB2477F000001110B9296284FEB11  |
| attr_list     | array | 属性数组 |    |

## attr_list

| 参数        | 类型   | 说明 | 示例 |
| ----------- | ------ | ---- | ---- |
| field_name     | string | 数据库字段 | memUsername   |
| attr_name   | string | 字段展示名称 | 姓名 |
| attr_value   | string | 数据库字段值 | wjfz |

## 返回参数

| 参数 | 类型   | 说明 | 示例                             |
| ---- | ------ | ---- | -------------------------------- |

## 返回结果示例

```json
{
  "status": "success",
  "httpCode": 200,
  "statusCode": 0,
  "data": {}
}
```
