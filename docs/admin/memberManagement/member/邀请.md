# 邀请

开发者：王俊锋泽

## 接口地址

`/member/v1/admin/member/invite`

## 请求方式

`POST`

## 请求参数

| 参数    | 类型  | 必填 | 示例                             |
| ------- | ----- | ---- | -------------------------------- |
| uids[0] | array |  否    | E18DB2477F000001110B9296284FEB11(指定邀请) |
| is_all | int |    否  |  1否，2是（批量邀请所有未加入）|
