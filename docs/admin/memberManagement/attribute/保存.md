#保存

开发者：王俊锋泽

## 接口地址
`/member/v1/admin/attribute/save`

## 请求方式
  `POST`

## 请求参数

| 参数        | 类型   | 必填 | 说明 |
| ----------- | ------ | ---- | ---- |
| attr_name | string | 是 |   |
| type | int | 是 | 属性类型：1=单行文本；2=多行文本；3=数字；4=日期；5=时间；6=日期时间；7=单选；8=多选；9=地址；10=图片；11=下拉框单选 |
| is_open_cp | int | 是 | 管理后台是否开启该字段, 0: 不开启; 1: 开启 |
| is_open | int | 是 | 是否开启：0=不开启；1=开启 |
| is_show | int | 是 | 是否显示：0=不显示；1=显示 |
| is_required_cp | int | 是 | 管理后台是否要求必填; 0: 非必填; 1: 必填 |
| is_required | int | 是 | 是否必填：0=非必填；1=必填 |
| is_allow_user_modify | int | 否 | 是否允许用户手机端编辑 0-不允许  1-允许 |
| is_allow_user_modify_edit | int | 否 | 是否允许编辑is_allow_user_modify字段：0=不允许；1=允许 |
| option | array | 否 | 属性选项 |
| attr_id | int | 否 |   |
