## 接口地址
`/member/v1/admin/setting/update`

## 请求方式
  `POST`

## 请求参数

`roleMode=select`

| 参数      | 类型   | 说明         | 示例     |
| --------- | ------ | ------------ | -------- |
| {{key}}   | string | 配置项       | roleMode |
| {{value}} | string | 新的配置项值 | select   |
