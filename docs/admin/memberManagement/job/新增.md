## 接口地址
`/member/v1/admin/job/add`

## 请求方式
  `POST`

## 请求参数

|参数|类型|是否必填|说明|
| - | - | - | - |
| jobName | string | 是 | 角色名称 |

## 返回参数

| 参数            | 类型    | 说明     | 示例                             |
| --------------- | ------- | -------- | -------------------------------- |
| jobId           | string  | 岗位id   | EF60B1067F00000109FA2B18A1F788CF |
| epId            | string  | 企业id   | 9BC78A067F0000017FE5C9821F85A4C0 |
| jobName         | string  | 岗位名称 | testttt                         |
| jobDisplayorder | integer | 排序     | 0                      |
