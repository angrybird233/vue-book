# 预算设置订单计算列表

开发者：王龙飞

## 接口地址

`/resource/v1/admin/budgetScheme/orderCalculateList`
### 改为
`/resource/v1/admin/budgetScheme/orderCalculate/list`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型 | 必填  | 描述 |
| - | - | - | - |
| budget_id | int | 是 | 预算方案ID |
| page | int | 否 | 当前页码 |
| page_size | int | 否 | 每页显示数量 |

## 返回参数

| 字段 | 类型 | 描述 |
| - | - | - |
|list|array|列表数据|
|meta|array|分页数据|

### list参数

| 字段 | 类型 | 描述 |
| - | - | - |
| id | int | 自增ID |
| classify | string | 分类名称(三级) |
| ratio | int | 比例 |
| start_time | int | 开始时间 |
| end_time | int | 结束时间 |

## meta参数

|参数|类型|说明|
| - | - | - |
|current_page|int|当前页|
|last_page|int|最后一页|
|page_size|int|每页展示数|
|total|int|总数|
