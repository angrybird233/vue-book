# 预算设置订单计算详情

开发者：王龙飞

## 接口地址

`/resource/v1/admin/budgetScheme/orderCalculateDetail`
### 改为
`/resource/v1/admin/budgetScheme/orderCalculate/detail`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型 | 必填  | 描述 |
| - | - | - | - |
| id | int | 是 | 自增ID |

## 返回参数

| 字段 | 类型 | 描述 |
| - | - | - |
| id | int | 自增ID |
| classify | string | 分类(三级) |
| ratio | int | 比例 |
| start_time | int | 开始时间 |
| end_time | int | 结束时间 |
