# 预算方案详情

开发者：王龙飞

## 接口地址

`/resource/v1/admin/budgetScheme/show`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型 | 必填  | 描述 |
| - | - | - | - |
| budget_id | int | 是 | 自增ID |

## 返回参数

| 字段 | 类型 | 描述 |
| - | - | - |
| budget_id | int | 预算方案主键id |
| budget_name | string | 预算方案名称 |
| type | int | 预算池类型 1订单计算2预估销量 |
| higher_limit | int | 上限 |
| lower_limit | int | 下限 |
