# 预算设置订单计算添加

开发者：王龙飞

## 接口地址

`/resource/v1/admin/budgetScheme/orderCalculateAdd`
### 改为
`/resource/v1/admin/budgetScheme/orderCalculate/add`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型 | 必填  | 描述 |
| - | - | - | - |
| budget_id | int | 是 | 预算方案ID |
| classify | string | 是 | 分类(三级) |
| ratio | int | 是 | 比例 |
| start_time | int | 是 | 开始时间 |
| end_time | int | 是 | 结束时间 |

## 成功响应

| 字段 | 类型 | 描述 |
| - | - | - |
| code | int | 状态码 |
| message | string | 返回信息 |
| data | array | 返回数据 |

## 返回参数

| 字段 | 类型 | 描述 |
| - | - | - |
无
