# 预算设置手动设置添加

开发者：王龙飞

## 接口地址

`/resource/v1/admin/budgetScheme/estimateSettingAdd`
### 改为
`/resource/v1/admin/budgetScheme/estimateSetting/add`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型 | 必填  | 描述 |
| - | - | - | - |
| budget_id | int | 是 | 预算方案ID |
| product_name | string | 是 | 产品名称 |
| ratio | int | 是 | 比例 |

## 返回参数

| 字段 | 类型 | 描述 |
| - | - | - |
无
