# 预算方案修改

开发者：王龙飞

## 接口地址

`/resource/v1/admin/budgetScheme/edit`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型 | 必填  | 描述 |
| - | - | - | - |
| budget_id | int | 是 | 主键ID |
| budget_name | string | 是 | 名称 |
| type | int | 是 | 预算池类型 1订单计算2预估销量  |
| higher_limit | int | 否| 上限 |
| lower_limit | int | 是| 下限 |

## 成功响应

| 字段 | 类型 | 描述 |
| - | - | - |
| code | int | 状态码 |
| message | string | 返回信息 |
| data | array | 返回数据 |

## 返回参数

| 字段 | 类型 | 描述 |
| - | - | - |
无
