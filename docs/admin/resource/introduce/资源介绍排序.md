# 资源介绍排序

开发者：王永国

## 接口地址

`/resource/v1/admin/introduce/sort`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| list | arr | 是 | 已排序资源介绍列表 |

## 请求参数 list
| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| introduce_id | int | 是 | 资源介绍id |

## 请求参数示例
```json
{
  "list": [
    {
      "introduce_id": 1,
      "introduce_name": "资源介绍名称"
    }
  ]
}
```

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status | str | 状态 |
| httpCode | int | http状态码 |
| statusCode | int | 状态码 |
| data | json | 返回数据 |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status | str | 状态 |
| httpCode | int | http状态码 |
| statusCode | int | 状态码 |
| message | str | 错误信息 |

| --- | --- | --- |
