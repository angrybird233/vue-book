# 资源介绍创建

开发者：王永国

## 接口地址

`/resource/v1/admin/introduce/create`

## 请求方式

`POST`

## 请求参数
| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| introduce_name | str | 是 | 资源介绍名称 |
| resource_id | int | 是 | 绑定资源库id |
| introduce_img | str | 是 | 资源介绍图片链接 |
| is_hot | int | 是 | 是否热门：1热门、2不热门 |
| content | str | 否 | 资源介绍详情 |

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status | str | 状态 |
| httpCode | int | http状态码 |
| statusCode | int | 状态码 |
| data | json | 返回数据 |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status | str | 状态 |
| httpCode | int | http状态码 |
| statusCode | int | 状态码 |
| message | str | 错误信息 |


## 返回结果预览

```json
{
  "status": "success",
  "httpCode": 200,
  "statusCode": 0,
  "data": {}
}
```
