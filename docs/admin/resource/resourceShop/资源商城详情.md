# 资源商城详情

开发者：胡彰正

## 接口地址

`/resource/v1/admin/resourceShop/detail`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型 | 必填 | 描述 |
| - | - | - | - |
| resource_shop_id | int | 是 | 商城id |

### 返回参数

| 字段 | 类型 | 描述 |
| - | - | - |
| resource_shop_id | int | 序号 |
| name | string | 商城名称 |
| goods_budget_classify | int | 商品消耗预算分类id |
| install_budget_classify | int | 安装费用消耗预算分类id |
