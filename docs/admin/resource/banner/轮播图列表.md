# 轮播图列表

开发者：王永国

## 接口地址

`/resource/v1/admin/banner/list`

## 请求方式

`POST`

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status | str | 状态 |
| httpCode | int | http状态码 |
| statusCode | int | 状态码 |
| data | json | 返回数据 |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status | str | 状态 |
| httpCode | int | http状态码 |
| statusCode | int | 状态码 |
| message | str | 错误信息 |

### data参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| banner_id | int | 轮播图id |
| banner_name | str | 轮播图名称 |
| banner_img | str | 轮播图图片链接 |

## 返回结果预览

```json
{
  "status": "success",
  "httpCode": 200,
  "statusCode": 0,
  "data": [
    {
      "banner_id": 1,
      "banner_name": "测试轮播图",
      "banner_img": "https://test-1.jpg",
    } 
  ]
}
```
