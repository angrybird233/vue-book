# 轮播图创建

开发者：王永国

## 接口地址

`/resource/v1/admin/banner/create`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| banner_name | str | 是 | 轮播图名称 |
| banner_img | str | 是 | 轮播图图片链接 |
| type | int | 是 | 轮播图内容类型：1超链接、2富文本 |
| content | str | 否 | 轮播图内容：超链接(type=1),富文本(type=2) |

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status | str | 状态 |
| httpCode | int | http状态码 |
| statusCode | int | 状态码 |
| data | json | 返回数据 |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status | str | 状态 |
| httpCode | int | http状态码 |
| statusCode | int | 状态码 |
| message | str | 错误信息 |
