# 登录

开发者：胡彰正

## 接口地址

`/admin/login`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| tempLoginToken | string |是| 临时令牌 | 

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | json  | 返回数据      |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

### data参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| user_id | int | ID |
| user_name | string | 姓名 |
| mobile | string | 手机 |
| role_name | string | 角色名称 |
| ep_name | string | 企业名称 |
| domain | string | domain，后续请求头部必传 |
| token | string | 令牌，后续请求头部必传 |
