# 核验手机

开发者：胡彰正

## 接口地址

`admin/checkMobilePwd`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| mobile | numeric | 是 | 手机 |
| password | string |是| 密码（md5一次） | 

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

### data参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| user_id | int | ID |
| user_name | string | 姓名 |
| mobile | string | 手机 |
| ep_name | string | 企业名称 |
| domain | string | domain |
| tempLoginToken | string | 临时令牌 |
