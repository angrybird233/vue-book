# 管理员菜单

开发者：胡彰正

## 接口地址

`/admin/menu`

## 请求方式

`POST`

## 请求参数

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

### data参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| permissions | string | 菜单(*表示超管,拥有所有权限) |
| role_type | int | 角色类型 1普通管理 2超级管理 超级管理拥有所有权限|
| role_name | string | 角色名称|
