# 管理员修改

开发者：胡彰正

## 接口地址

`/admin/user/edit`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| user_id | int | 是 | 管理员ID |
| user_name | string |是| 姓名 | 
| email | string | 是|邮箱 |
| mem_uid | string |否| 绑定人员id |
| mem_face | string |否| 绑定人员头像 |
| mem_username | string |否| 绑定人员姓名 |
| login_status | string |是| 是否允许登录1禁止2允许 |
| role_id | int | 是|角色id |

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | json  | 返回数据      |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

## 返回结果预览

```json
{
  "status": "success",
  "httpCode": 200,
  "statusCode": 0,
  "data": {}
}
```
