# 白名单导出

开发者：段昭辉

## 接口地址

`/admin/white/export`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| user_name | string | 否 | 名称 |
| mobile | string | 否 | 手机号 |
| start_time | int | 否 | 加入开始时间（13位时间戳） |
| end_time | int | 否 | 加入结束时间（13位时间戳） |

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |
