# 忘记密码_返回账户列表

开发者：胡彰正

## 接口地址

`/admin/forgetPwd/userList`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| mobile   | string    | 是  | 手机号   |
| key | string | 是 | 短信验证码key |
| captcha | string |是| 短信验证码 ||

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

#### data参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| corp_name | string | 企业名称 |
| domain | string | 企业标识 |
