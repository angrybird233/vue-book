# 忘记密码_重置密码

开发者：胡彰正

## 接口地址

`/admin/forgetPwd/initPwd`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| mobile   | string    | 是  | 手机号   |
| key | string | 是 | 短信验证码key |
| captcha | string |是| 短信验证码 ||
| password | string |是| 密码(md5一次) ||

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

## 返回结果预览

```json
{
  "status": "success",
  "httpCode": 200,
  "statusCode": 0,
  "data": {}
}
```
