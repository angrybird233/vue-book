# 验证短信验证码

开发者：胡彰正

## 接口地址

`/admin/common/verifySmsCode`

## 请求方式

`POST`

## 请求参数

| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| key   | string    | 是  | 短信key   |
| captcha   | string    | 是  | 验证码   |
| is_del   | int    | 否  | 是否删除验证码，1否 2是（不传默认2）   |

## 成功响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | json  | 返回数据      |

## 失败响应

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

### data参数

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| is_pass | bool | true通过 false未通过 |

## 返回结果预览

```json
{
  "status": "success",
  "httpCode": 200,
  "statusCode": 0,
  "data": {
    "is_pass": true
  }
}
```
