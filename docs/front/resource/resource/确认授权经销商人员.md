## 请求方式
`POST`

## 请求参数
| 字段 | 类型 | 必填  | 描述 |
| - | - | - | - |
| resource_id | int | 是 | 资源ID |
| store_code | string | 是 | 门店code |
| uid | string | 是 | 成员UID |

| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |

| requestId | string  | 请求ID |
