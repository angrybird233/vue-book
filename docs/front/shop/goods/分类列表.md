#分类列表

开发者：

## 接口地址
`goods/v1/front/category/index`

## 请求方式

  `GET`
  
## 请求参数
| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| pool_id | 是 | int | 池子id |
| status | 是 | int | 状态:1关闭 2开启（默认为2） |

## 成功响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

### data参数
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| cate_id | int | 分类id |
| parent_id | int | 父类Id |
| cate_name | String | 分类名称 |
| status | int | 是否开启：1关闭  2开启 |
| sub_cate | array | 子类数组 |
