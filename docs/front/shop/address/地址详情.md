#地址详情接口

开发者：王龙飞

## 接口地址
`/shop/v1/front/area/info` `GET`
  
## 请求参数

|参数|类型|是否必填|说明|
| - | - | - | - |
| address_id | 否 | int | 地址主键id 不传id则返回默认地址|
