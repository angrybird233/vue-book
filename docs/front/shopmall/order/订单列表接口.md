#订单列表接口

开发者：王龙飞

## 列表接口地址
`/shopmall/v1/front/order/list`  `POST`
## 列表接口地址
`/ShopMall/Api/Order/MyOrder`  `POST`

## 请求参数

|参数|类型|是否必填|说明|
| - | - | - | - |
|<font color=#ff0000 >`order_recycle`</font> | 否 | String | 是否回收站 1 订单列表 2 回收站（不传默认未订单列表） |
| order_status | 是 | Int | 订单状态 0 全部 ， 1 待付款 ， 2 代发货 ， 3 待收货， 4 已签收， 5 待确认 ， 6 已取消 ， 7 等待支付确认 , 取消待确认 |
| start_time | 否 | String | 开始时间 |
| end_time | 否 | String | 结束时间 |
| keywords | 否 | String | 搜索keywords |
| cate_id | 否 | String | 商城入口id（二级商城属性id） |
| consignee_name | 否 | String | 收货人姓名 |
| page | 否 | Int | 当前页 |
| page_size | 否 | Int | 当前页条数 |

## 返回参数

|参数|类型|说明|
| - | - | - |
| meta | array | 分页数据 |
| list | array | 属性名称 |

## meta 返回参数
|参数|类型|说明|
| - | - | - |
| current_page | Int | 当前页码 |
| last_page | int | 最后一页页码 |
| page_size | int | 每页查询条数 |
| total | int | 数据总数 |

## list 返回参数
|参数|类型|说明|
| - | - | - |
| order_id | Int | 订单id |
| order_sn | Int | 订单编号 |
| jd_order_sn | Int | 第三方订单编号 |
| supplier_type | Int | 供应商类型 1 京东慧采 2 网易 3 自营 4 苏宁 |
| supplier_name | String | 供应商名称 |
| pay_type | int | 付款方式 1 线下付款 2 货到付款 3 积分兑换 |
| order_status | int | 订单状态 1 待付款 ， 2 代发货 ， 3 待收货， 4 已签收， 5 待确认 ， 6 已取消 ， 7 等待支付确认 , 9 ， 取消待确认 |
| pay_status | Int | 支付状态 ： 1 已付款 ， 2 待付款 |
| pause_status | Int | 暂停状态 1 未暂停 2 已暂停 |
| is_jd_cancel | Int | 是否京东取消 ： 1 京东取消 ， 2 客户取消 ， 0 未取消 （order_stat = 6 或者 0 时返回） |
| is_voucher | Int | 是否上传凭证 ： 1 已上传 ， 2 未上传 |
| submit_state | Int | 是否确认下单 ： 1 已确认 ， 2 未确认 （之前名称 ： submitState） |
| invoicetype | int | 发票类型 2 增票 3 普通电子发票 |
| instate | int | 发票状态 1 未开票 2 已开票 |
| created | string | 下单时间 |
| quantity | string | 订单中商品总数 |
| money | string | 订单中交易总积分 |
| company_name | string | 户名 |
| bank_account | string | 银行账号 |
| open_bank | string | 开户银行 |
| rule_name | string | 规则名称 |
| rule_status | int | 规则状态 1 已失效 2 未失效 |
| apply_type | int | 申请状态 1 已申请 2 未申请 |
| express_no | string | 运单号 |
| express_com | string | 物流公司名称 |
| company_code | string | 物流公司编码 |
| goods_list | array | 订单中商品列表 |

## goods_list 返回数据
|参数|类型|说明|
| - | - | - |
| goods_id | int | 商品id |
| sku_code | String | 商品skucode |
| goods_name | string | 商品名称 |
| goods_picture | string | 商品图片 |
| spec_value | string | 商品规格 |
| goods_quantity | string | 申请的商品的数量 |
| goods_money | string | 申请的商品的积分 |
