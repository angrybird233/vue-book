#订单删除/恢复接口

开发者：王龙飞

## pc端接口地址
`/shopmall/v1/front/order/orderDelOrRec`  `POST`
## 企微h5端接口地址
`/ShopMall/Api/Order/OrderDelOrRec` `POST`

## 请求参数

|参数|类型|是否必填|说明|
| - | - | - | - |
|order_id|int|是|订单主键id|
|del_status|int|是|订单删除状态 1 恢复订单 2 删除订单|

| - | - | - |
