#供应商列表

开发者：王秀静

## 接口地址
`goods/v1/front/supplier/index`

## 请求方式

  `GET`
  
## 请求参数

|参数|类型|是否必填|说明|
| - | - | - | - |
| pool_id | 是 | int | 池子id |


## 返回参数
|参数|类型|说明|
| - | - | - |
| supplier_id | int | 供应商id |
| supplier_name | String | 供应商名称 |
