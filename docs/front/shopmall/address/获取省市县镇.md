#获取省市县镇地址接口

开发者：王龙飞

## smartpal-pc端 接口地址
`/goods/v1/front/area/index` `GET`
## smartpal企微h5端 接口地址
`/GoodsLibrarys/Api/Pool/Area` `POST`
  
## 请求参数

|参数|类型|是否必填|说明|
| - | - | - | - |
| area_id | 否 | int|获取省不必填，其他都为必填|
| level | 是 | int|1省,2 市,3县/镇,4街道|
|<font color=#ff0000 >`supplier_type`</font> | 否 | int|供应商类型 默认 1 京东 3 自营 4 苏宁|

## 返回参数

|参数|类型|说明|
| - | - | - |
| area_id | int |数据主键|
| parent_id | int |父级id|
| area_name | String |地区名称|
| area_code | int |地区编码，为空时代表京东没有地区|
| level | int |1省，2市，3县（区），4镇|
