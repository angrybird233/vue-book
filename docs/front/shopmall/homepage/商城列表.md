#商城列表接口

开发者：王龙飞

## 接口地址
`/shopmall/v1/front/ShopConfig/shopList`

## 请求方式
  `GET`
  
## 请求参数

|参数|类型|是否必填|说明|
| - | - | - | - |

## 返回参数
|参数|类型|说明|
| - | - | - |
| name | String | 商城名称 |
| cate_id | String | 商城入口id（商品属性二级分类） |
| total_integral | String | 总积分 |
| used_integral | String | 已使用积分 |
| type | String | 1垫付 2兑换 |
| cover_url | String | 图片url |
| pool_id | int | 子商品池id |
| failure_date | String | 失效时间 |
| invalid_integral | String | 即将失效的积分 |
| goods_state | Int | 商品状态 1：上架  2：下架 |
| activity | Int | 是否有活动 0 =无 1=有 |
| is_watch | Int | 是否有红点 1 = 无 2=有 |
