#可用积分及购物车数量

开发者：王秀静

## 接口地址
`/shopmall/v1/front/cart/integralNumber`

## 请求方式

  `GET`
  
## 请求参数

|参数|类型|是否必填|说明|
| - | - | - | - |
| cate_id | 是 | int | 入口分类id |


## 返回参数
|参数|类型|说明|
| - | - | - |
| pool_is_open | int | 分类是否在有效期   1是   2否 |
| integral | String | 可用积分 |
| number | String | 购物车数量 |
