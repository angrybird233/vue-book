#积分列表

开发者：

## 接口地址
`integralRule/v1/front/integral/list`

## 请求方式

  `POST`
  
## 请求参数

|参数|类型|是否必填|说明|
| - | - | - | - |
| shop_id | 否 | int | 商城id |


## 返回参数

|参数|类型|说明|
| - | - | - |
| rule_id | int | 规则主键 |
| rule_code | int | 规则编码 |
| rule_name | string | 规则名称 |
| failure_date | int | 失效时间 |
| available_integral | int | 剩余可用积分 |
| total_integral | int | 总积分 |
| is_expire | int | 是否过期 1是 2否 |


## 返回数据示例
