#商城列表接口
开发者：王龙飞

## 接口地址
`/shopCustomize/v1/front/homepage/shopList`

## 请求方式
  `GET`
  
## 请求参数
| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |

## 成功响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

### data参数
| 字段 | 类型 | 描述 |
| --- | --- | --- |
| shop_name | String | 商城名称 |
| shop_id | int | 商城主键id |
| total_integral | String | 总积分 |
| used_integral | String | 已使用积分 |
| shop_type | String | 商城类型 1兑换 2 垫付|
| cover_img | String | 商城封面图片url |
| pool_id | int | 商品池ID |
| failure_date | String | 失效时间 |
| invalid_integral | String | 即将失效的积分 |
