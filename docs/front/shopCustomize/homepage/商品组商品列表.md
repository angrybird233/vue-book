#banner文章详情接口

开发者：王龙飞

## 接口地址
`/shopCustomize/v1/front/homepage/groupGoodsList`

## 请求方式
  `POST`
  
## 请求参数
| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| goods_group_id | 是 | int | 商品组主键id |

## 成功响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

### data参数
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| list | array | 商品列表 |
| meta | array | 分页数据 |

## meta 返回参数
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| current_page | Int | 当前页码 |
| last_page | int | 最后一页页码 |
| page_size | int | 每页查询条数 |
| total | int | 数据总数 |

## list 返回参数
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| spu_id | int | 商品的主键 |
| shop_id | int | 商城主键id |
| spu_code | string | spu code |
| supplier_type | int | 数据来源 1京东，2网易，3自营，4自营 |
| product_name | string | 商品名称 |
| supplier_code | int | 供应商code |
| supplier_name | string | 供应商名称 |
| cover_img | string | 商品主图url |
| order_count | int | 销量 |
| is_stock | int | 有无库存 1 有 2 无 0 未确定 |
| is_shelf | int | 上下架状态 1下架,2上架 |
| price | number | 价格 |
