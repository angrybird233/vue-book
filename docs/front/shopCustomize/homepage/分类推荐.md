#分类推荐列表接口
开发者：王龙飞

## 接口地址
`/shopCustomize/v1/front/homepage/recommendClass`

## 请求方式
  `POST`
  
## 请求参数
| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| shop_id | 是 | int | 商城主键id |

## 成功响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

### data参数
| 字段 | 类型 | 描述 |
| --- | --- | --- |
| recommend_class_id | int | 主键ID |
| img_url | String | 图片链接 |
| recommend_class_name | String | 名称 |
| cate_one | string | 一级分类id |
| cate_two | string | 二级分类id |
