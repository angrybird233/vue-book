#可用积分和购物车商品数量
开发者：王龙飞

## 接口地址
`/shopCustomize/Api/homepage/IntegralNumber` `POST`

## 请求参数
| 字段 | 类型   | 必填 | 描述     |
| ---- | ------ | ---- | -------- |
| shop_id | int | 是 | 商城主键id |

## 成功响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| data  | array  | 返回数据      |

## 失败响应
| 字段       | 类型    | 描述        |
| ---------- | ------- | ----------- |
| status    | string  | 状态    |
| httpCode     | int  | http状态码    |
| statusCode | int  | 状态码 |
| message  | string  | 错误信息      |

### data参数
| 字段 | 类型 | 描述 |
| --- | --- | --- |
| integral | String | 可用积分 |
| number | String | 购物车内商品数量 |
| goods_class | String | 购物车内商品种类数量 |
