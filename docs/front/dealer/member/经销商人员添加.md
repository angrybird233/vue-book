#经销商人员更新

开发者：王俊锋泽

## 接口地址
`/dealer-management/v1/front/member/add`

## 请求方式
  `POST`

## 请求参数

| 参数    | 类型   | 说明                            |
| ------- | ------ | ------------------------------- |
|mem_name|string|姓名|
|mem_tel|string|手机号|
|dealer_code|string|经销商编码|
|position|string|职业角色|
|channel|string|渠道|
|label_ids[0]|string|标签数组|
|vehicle|string|车牌号|

