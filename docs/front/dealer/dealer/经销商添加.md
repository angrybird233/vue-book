#经销商添加接口

开发者：王俊锋泽

## 接口地址
`/dealer-management/v1/front/dealer/add`

## 请求方式
  `POST`

## 请求参数

| 参数        | 类型   | 说明       |
| ----------- | ------ | ---------- |
| dealer_code | string | 经销商编码 |
| dealer_name | string | 经销商名称 |

|参数|类型|说明|
| - | - | - |
|      |      |      |
