#经销商列表接口

开发者：王俊锋泽

## 接口地址
`/dealer-management/v1/front/dealer/list`

## 请求方式
  `POST`

## 请求参数

|参数|类型|是否必填|说明|
| - | - | - | - |
| type | int | 是 | 1:营业中；2:待确认；3:已关闭 |
| page | int | 否 |  |
| page_size | int | 否 |  |


## 返回参数
|参数|类型|说明|
| - | - | - |
| list | array | 数据列表 |
| total | int | 总数 |
| layer | int | 权限字段 |
| is_dsr | int | 是否是dsr |

## list 返回参数
|参数|类型|说明|
| - | - | - |
| dealer_code | string | 经销商code |
| dealer_name | string | 经销商名称 |
